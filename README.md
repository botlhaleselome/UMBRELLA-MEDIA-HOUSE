## 📖 About

The **Umbrella Media House Ecosystem** is an ambitious full-stack platform that combines two complementary brands under one digital ecosystem:

### 🎙️ LiftedVeilMedia

An independent publication exploring:

* Philosophy
* Society
* Artificial Intelligence
* Politics
* Digital Culture
* Future Technology
* Documentary Journalism
* Long-form Essays

### 💻 FluenyTech

A modern technology education platform covering:

* Artificial Intelligence
* Prompt Engineering
* Linux
* Programming
* Cybersecurity
* Software Engineering
* Cloud Computing
* Automation
* Emerging Technologies

Together, these platforms create a unique blend of **critical thinking** and **technical empowerment**.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/UMBRELLA-MEDIA-HOUSE.git
```

Navigate into the project

```bash
cd UMBRELLA-MEDIA-HOUSE
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
